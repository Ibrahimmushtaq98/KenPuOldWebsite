Tensor representation of data¶

We want to represent some sort of data DD using a tensor xx. The representation can be seen as two functions:

enc:Data→Tensor:D↦xenc:Data→Tensor:D↦x

such that there exists a decoding function:

dec:Tensor→Data:x↦Ddec:Tensor→Data:x↦D

where

dec(enc(D))=Ddec(enc(D))=D

Variables over single categorical value¶

A categorical variable is one whose value must be drawn from a finite set.

For example,

color∈{red,blue,green}color∈{red,blue,green}

In [0]:
COLORS = ['RED', 'BLUE', 'GREEN']

fav_color = COLORS[0]

print(fav_color)

RED

We need an encoding of categorical variables with the following.

  • A proper encoding (with a decoding function).
  • Each encoding vector are equally spaced.

The solution is called the one hot encoding.

enc:Domain→R|Domain|enc:Domain→R|Domain|

In [0]:
def enc_cat(domain, value):
  v = np.zeros(len(domain))
  i = domain.index(value)
  v[i] = 1.
  return v
In [0]:
def dec_cat(domain, vector):
  i, = np.where(vector > 0)
  return domain[i[0]]

Text¶

There are several ways of capturing text, each with its own advantages and disadvantages.

  • Text as a bag of words
  • Text as a sequence of words

Text as bag of word¶

Suppose we have a text:

text = "we don't know what we don't know"

As a bag of words, we record the distinct words and their frequences, but not the sequences of occurrences of thes words.

This allows us to encode bag of words as a single vector with dimensionality of |Voc||Voc|.

In [0]:
def bag_words(text):
  bag = dict()
  for w in text.split():
    if w in bag:
      bag[w] += 1
    else:
      bag[w] = 1
  return bag
In [0]:
def enc_bag(voc, bag):
  n = len(voc)
  v = np.zeros(n)
  for w, freq in bag.itemize():
    v += enc_word(voc, bag) * freq
  return v
