def plot_2d_vectors(*E, **F):
  names = E[::2]
  vectors = E[1::2]
  for (name, vec) in zip(names, vectors):
    pl.arrow(0, 0, vec[0], vec[1], **F)
    pl.text(vec[0], vec[1], name)
  V = np.abs(np.array(vectors))
  v_lengths = np.max(V, axis=0) * 2
  pl.xlim(-v_lengths[0], v_lengths[0])
  pl.ylim(-v_lengths[1], v_lengths[1])

Vector operations¶

Let's look at some vector specific operations beyond the basic arithmetics

Inner product¶

$$ \left<\mathbf{u}, \mathbf{v}\right> = \sum_i u_i\cdot v_i $$

Length of vector¶

The length of a vector, $|\mathbf{v}|$, is the distance from the origin. This is given by:

$$|\mathbf{v}|^2 = \left<v, v\right>$$
In [0]:
print('|u| = %.2f = %.2f' % (np.sqrt(np.dot(u, u)), np.linalg.norm(u)))
print('|v| = %.2f = %.2f' % (np.sqrt(np.dot(v, v)), np.linalg.norm(v)))

Angle between vectors¶

The angle between two vectors can be calculated using the inner product operation.

$$ \cos(\theta) = \frac{\left<\mathbf{u}, \mathbf{v}\right>}{|\mathbf{u}|\cdot|\mathbf{v}|} $$

Orthogonality¶

Two vectors are orthogonal if the angle between them are 90 degrees. This means that:

$$ \cos(\theta) = 0 $$

Equivalently: $\left<\mathbf{u}, \mathbf{v}\right> = 0$

Cross product in 3D¶

In the vector space $\mathbb{R}^3$, there is a vector operation that can compute an orthogonal vector from two other vectors.

$$ \mathbf{u}\times\mathbf{v} = \left[\begin{array}{c} u_2v_3-u_3v_2 \\ u_3v_1-u_1v_3 \\ u_1v_2-u_2v_1 \end{array}\right]$$

def cross_product(u, v):
    w1 = u[1]*v[2] - u[2]*v[1]
    w2 = u[2]*v[0] - u[0]*v[2]
    w3 = u[0]*v[1] - u[1]*v[0]
    return np.array([w1, w2, w3])

Planes¶

Planes are sets of points that are linear combinations of two vectors. The two vectors must not be co-linear.

$$ P = \{a\mathbf{u} + b\mathbf{v} + \mathbf{w} : a, b\in \mathbb{R}\}$$

Projections and distances¶

Distances¶

What is the distance between two vectors:

  • $d(\mathbf{u}, \mathbf{v}) = |\mathbf{u}-\mathbf{v}|$

What is the distance between a vector $\mathbf{u}$ and a line $L$?

  • We need to compute the projection of $\mathbf{u}$ on to the line $L$: $\mathrm{proj}(\mathbf{u}, L)$
  • Then compute the distance $d(\mathbf{u}, \mathrm{proj}(\mathbf{u}, L))$

Distances¶

What is the distance between a vector $\mathbf{x}$ and a line $L$?

  • We need to compute the projection of $\mathbf{x}$ on to the line $L$: $\mathrm{proj}(\mathbf{x}, L)$
  • Then compute the distance $d(\mathbf{x}, \mathrm{proj}(\mathbf{x}, L))$

Distances: vector to plane¶

What is the distance between a vector $\mathbf{x}$ and a plane $P$?

  • We need to compute the projection of $\mathbf{x}$ onto the plane P: $\mathrm{proj}(\mathbf{x}, P)$
  • Then compute the distance $d(\mathbf{x}, \mathrm{proj}(\mathbf{x}, P))$

Projection onto a line¶

  • Let a line be defined by its direction vector $\mathbf{u}$, where $|\mathbf{u}| = 1$.

How do we project $\mathbf{x}$ to the line $L(\mathbf{u})$? Let $\mathbf{y} = \mathrm{proj}(\mathbf{x}, L(\mathbf{u}))$

Observe:

  • $\mathbf{y}$ is colinear to $\mathbf{u}$.
  • $$|\mathbf{y}| = |\mathbf{x}|\cos(\theta) = |\mathbf{x}| \frac{\left<\mathbf{x},\mathbf{u}\right>} {|\mathbf{x}| |\mathbf{u}|} = \left<\mathbf{x}, \mathbf{u}\right>$$

Thus,

$$ \mathbf{y} = \left<\mathbf{x}, \mathbf{u}\right> \mathbf{u} $$

Projection onto a plane¶

Consider a vector $\mathbf{x}$.

Let a plane be defined by its two directional vectors $\mathbf{u}$ and $\mathbf{v}$. So, we write the plane as $P(\mathbf{u}, \mathbf{v})$.

  • First we compute the normal vector $\mathbf{n} = \mathbf{u}\times\mathbf{v}$.

  • We project the vector onto $L(\mathbf{n})$, and obtain $\mathbf{y_1} = \mathrm{proj}(\mathbf{x}, L(\mathbf{n}))$.

  • Now we can compute the projection onto the plane as: $$\mathrm{proj}(\mathbf{x}, P) = \mathbf{x} - \mathrm{proj}(x, L(\mathbf{n}))$$

  • The distance is given: $$d(\mathbf{x}, P) = |\mathrm{proj}(x, L(\mathbf{n}))|$$
