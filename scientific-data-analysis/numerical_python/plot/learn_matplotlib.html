Note:

A requirement of fill_between(xs, y1, y2) is that the two lines y1 and y2 must have the same x-values, as given by xs.

If we have two lines given by two very different x-values, we can use the interpolation function to make them the same.

  2. We (somehow) do not have the functions that generated the two lines.

Contour Plots

Consider a 2D grid

(x1,y1) (x2,y1) (x3,y1) ... (xn, y1)
(x1,y2) (x2,y2) (x3,y3) ... (xn, y2)
.
.
.
(x1,ym) (x2,ym) (x3,ym) ... (xn,ym)

At each coordinate (xi,yj), we can have height z[i,j].

Contour plot visualizes the heights as a 2D plot.
